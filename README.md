## The feature I did
* Lets you play tic-tac-toe
* Indicates when a player has won the game,
* Restart the game

## Available Scripts
In the project directory, you can run:

``` bash
# Runs the app in the development mode.
yarn start

# Builds the app for production
yarn build
```
